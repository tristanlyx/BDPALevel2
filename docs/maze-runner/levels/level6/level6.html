Level 6

  • This level introduces the blue "password" box. In order to get past this box, you will need to make cabbage man say the correct password
  • Start out by defining this function:
    function runLevel(){
      var cabbageMan = new MazeRunner.CabbageMan();
      cabbageMan.moveRight();
     
      cabbageMan.moveRight(18);
    }
  • You'll notice that you can't yet get past the blue block. This particular blue bock requires that you take the number inside and say "BIG" if it's bigger than five, and "SMALL" if it's less than or equal to five
  • Define a definePassword function like this. prompt will be the value of whatever's in the blue box
    function decidePassword(prompt){
     if(prompt > 5){
        return "BIG";
     } else {
       return "SMALL";
     }
    }
  • Finally we need to tell cabbageMan that he should use the decidePassword function after he moves right. We do this by calling the calculatePassword function. Add this line of code as the third line of your runLevel function:
    cabbageMan.calculatePassword("RIGHT", decidePassword);
    This tells cabbageMan to calculate the password for the box on the "RIGHT" using the decide password function.
