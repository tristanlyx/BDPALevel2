Level 17

  • The password is changed!!!
  • In this level the prompt is an Array with two numbers.
  • Your decidePassword will compare the two numbers to see which one is bigger
  • If the 0th number is bigger, it should return 0. If the 1th number is bigger, it should return 1
  • If the two numbers are the same size, then you should return "EQUAL"
  • HINT: You can access the 0th element of the array like this: prompt[0]
  • HINT: You can access the 1th element of the array like this: prompt[1]
